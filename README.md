# 📚 BookEduPuc - Sistema de Biblioteca Escolar

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como parte do MVP da disciplina de Gestão Ágil de Projetos e Produtos da PUC.

O objetivo é propor uma solução digital para o gerenciamento de bibliotecas escolares, permitindo o controle de alunos, livros, empréstimos, devoluções e reservas, utilizando conceitos de Engenharia de Requisitos, Gestão Ágil de Produtos, Gestão Ágil de Projetos e Design de Interface.



## 🎯 Objetivo do MVP

Desenvolver uma plataforma simples e intuitiva para auxiliar bibliotecários e alunos no gerenciamento de acervos e empréstimos de livros.

## 🚀 Funcionalidades do MVP

### Gestão de Usuários

* Login no sistema
* Cadastro e consulta de alunos

### Gestão de Livros

* Cadastro de livros
* Pesquisa de livros
* Consulta de disponibilidade

### Gestão de Empréstimos

* Registro de empréstimos
* Registro de devoluções
* Reserva de livros
